Acerca de este sitio

Este sitio web ha sido diseñado para ser visto a una resolución de 800 x 600 como mínimo, con color de 16 bits.

Si bien, el sitio puede ser visto con cualquier "browser", se recomienda el uso de uno que sea capaz de manipular y ejecutar correctamente los lenguajes de programación Java® y JavaScript®, para poder apreciarlo eficientemente.

Este Sitio WEB podría incluir algunos documentos de referencia en formato PDF. Para poder apreciar y manipular estos documentos se requiere el programa Acrobat Reader.
